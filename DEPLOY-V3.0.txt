DEPLOYMENT:
1. Delete the old website files from the GitHub repository.
2. Upload the CONTENTS of this ZIP directly to the repository root.
3. Confirm index.html, CNAME, assets/, robots.txt, and sitemap.xml are at the root.
4. Wait for GitHub Pages deployment.
5. Hard refresh with Ctrl + Shift + R.
